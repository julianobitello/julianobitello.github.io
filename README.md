# 🚀 Portfólio Front-End

Este é meu portfólio pessoal desenvolvido para apresentar meus projetos, habilidades e experiência como desenvolvedor Front-End.

## 📌 Sobre o projeto

O objetivo deste projeto é centralizar meus trabalhos e facilitar a visualização das minhas habilidades em desenvolvimento web, criação de interfaces e consumo de APIs.

O portfólio é focado em performance, responsividade e boas práticas de desenvolvimento moderno.

---

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3 / TailwindCSS (se usar, ajuste aqui)
- JavaScript (ES6+)
- React.js (ou outro framework, se estiver usando)
- TypeScript (se aplicável)
- Vite / Next.js (se aplicável)
